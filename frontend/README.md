# Mini-OpenClaw Frontend

这是 Mini-OpenClaw 的前端部分，基于 Next.js 构建。

## 快速开始

1.  安装依赖:
    ```bash
    npm install
    ```

2.  启动开发服务器:
    ```bash
    npm run dev
    ```

3.  访问 `http://localhost:3000`。

## 构建生产版本

1.  构建:
    ```bash
    npm run build
    ```

2.  启动:
    ```bash
    npm start
    ```
